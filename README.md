# Book App

Aplikasi mobile sederhana untuk mencari buku anak-anak, melihat detail informasi buku, dan menyimpan daftar buku favorit secara offline. Proyek ini dibuat sebagai bagian dari tantangan seleksi Mobile Developer di **Ziyad** (Penerbit Buku Anak & Edukasi Terbesar di Indonesia).

---

## ✨ Fitur Utama

* **Daftar Buku Anak:** Menampilkan 20 buku anak terbaru langsung dari Open Library API lengkap dengan gambar sampul, judul, penulis, dan tahun terbit.
* **Pencarian Realtime:** Ketik judul buku yang dicari, dan daftar buku akan langsung tersaring otomatis tanpa perlu menekan tombol cari.
* **Detail Buku Lengkap:** Klik pada buku untuk melihat informasi mendalam seperti bahasa, jumlah edisi, status baca online, hingga akses gratis/pinjam.
* **Daftar Favorit Offline:** Simpan buku yang kamu sukai dengan menekan tombol hati. Data tetap aman tersimpan di memori HP meskipun aplikasi ditutup atau HP dimatikan.
* **Indikator Loading & Error:** Dilengkapi animasi loading saat memuat data dan pesan peringatan yang ramah jika koneksi internet terputus.

---

## 🛠️ Teknologi yang Digunakan

* **Framework:** React Native (Expo SDK 54)
* **Bahasa:** JavaScript / React 19
* **Penyimpanan Lokal:** AsyncStorage (Untuk fitur Favorit Offline)
* **Navigasi antar Halaman:** React Navigation Stack
